## 그래서 Crerty가 갑자기 닫힌 이유는 무엇이고, 왜 복구를 하지 않았습니까?
위에서 말씀드렸듯 학업 문제로 매일 접속하던 사이트가 2일, 7일, 14일씩 접속을 하지 않더니 아예 한 달 동안 사이트에 단 한 번도 못 들어가는 일도 많았고, 사이트는 온갖 스팸과 반달로 난장판이 되었습니다. 도저히 손을 쓸 수 없어서 한번 백섭을 진행하기도 했고요. 어쨌든, 저는 서버비 결제 시 자동 결제를 사용하지 않고 매 달 수동으로 결제했는데, 서버 결제하는 것 까지 신경을 못 써서 서버비가 계속 연체되었고, 결국 계정과 서버가 강제 삭제되면서 자동으로 닫혔습니다.

지금까지 복구를 못 한 이유는 간단합니다. Vultr를 사용하면서 카드 도용 피해를 2번이나 당했습니다. 한 번은 실제로 결제가 되었고 한 번은 다행히 카드사에서 이상거래를 탐지해서 결제는 막았습니다. 하지만 이 사건 이후 카드를 정지해 둔 상태입니다. 재발급을 받아야 하는데 그럴 시간이 없었습니다. 제가 해외 결제가 되는 카드가 그 카드 하나 뿐이라 다른 서버도 이용 못하고, 오라클 프리 티어도 가입 시 해외 결제 지원 카드를 요구해 전혀 손을 못 쓰고 있었습니다.

참고로 계정과 서버가 강제 삭제되었어도 삭제 몇일 전 백업을 해 놓아서 DB와 엔진은 모두 멀쩡합니다.
